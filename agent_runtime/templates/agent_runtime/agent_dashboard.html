1. 采集调度 Celery beat 按来源类型触发抓取，失败源进入退避和重试队列。
2. 抽取与清洗 使用规则抽取、正文清洗、OCR 和发布时间解析，减少网页噪声。
3. 去重与发布 内容指纹和近重复检测合并重复来源，只公开审核通过的摘要。
4. RAG 索引 已发布内容切分为 chunk，同步到 Meilisearch，保留来源和详情页引用。
5. 问答生成 检索上下文后调用 DeepSeek；预算不足或模型异常时转为零成本检索式回答。
6. 观测与自愈 AgentRun / AgentStep 记录状态、耗时、工具名、费用和低成本自愈动作。
